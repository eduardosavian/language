# The Simplex Language or Slang

Made by Eduardo Savian, Pablo Marques e Marcos Fehlauer

## Setup on Unix

### Give permition to the bash script

```bash
chmod +x generate.sh
```

### Generate ANTLR files

```bash
./generate.sh
```

## Compile and run wit **Ant**

### Ant build

```bash
ant build
```
### Ant run

```bash
ant -Darg="tests/<input_file>.txt" run
```
