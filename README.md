# The Simplex Language or Slang

Made by Eduardo Savian, Pablo Marques e Marcos Fehlauer

## Setup on Unix

### Give permitions to the bash scripts

```bash
chmod +x generate.sh compile.sh run.sh src/test.sh
```

### Generate Parser

```bash
./generate.sh
```

### Change directory to source

```bash
./compile.sh
```
## Run or test

### Run the ANTLR

```bash
./run.sh
```

### Test the ANTLR

```bash
cd src/ && ./test.sh
```

```bash
./test.sh
```

```bash
cd ..
```
