# Simple Language

## Setup on Unix

### Give permitions to the bash scripts

```bash
chmod +x antlr.sh src/grun.sh
```

### Generate Parser

```bash
./antlr.sh
```

### Change directory to source

```bash
cd src/
```

### Run the ANTLR Test

```bash
./test Simplex <flags>
```

### Back to the root directory

```bash
cd ..
```
