# Simple Language

## Setup on Unix

### Give permitions to the bash scripts

```bash
chmod +x generate.sh compile.sh run.sh
```

### Generate Parser

```bash
./generate.sh
```

### Copy the template Main.java from template/ to src/

```bash
-
```

### Change directory to source

```bash
./compile.sh
```

### Run the ANTLR Test

```bash
./run.sh
```
