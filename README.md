# Simple Language

## Setup on Unix

```bash
chmod +x antlr4.sh grun.sh
```

```bash
./antlr4.sh src/Simplex.g4
```

```bash
javac src/*.java
```

```bash
cd src/
```

```bash
./grun Simplex r -tokens
``.sh`
